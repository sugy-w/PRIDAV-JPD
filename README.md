# Semestral project Data Science Principles (team JPD)

Repository accumulates all the codes used through the project for the subject PRIDAV, during the Winter Semester of 2025/2026 at the Faculty of Mathematics, Physics and Informatics Comenius University in Bratislava.

In the project we focus on deeply analysing data from the bicycle traffic counters in the city of Bratislava, with connection to the weather data. We used primarily ML techniques to determined effect of using such data in order to predict total number of intercepts on each of the roads with daily frequency.
